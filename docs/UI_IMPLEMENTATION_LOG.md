# UI 实施记录

本文档记录 UI 相关的具体落地调整，避免只依赖聊天记录。设计原则见 [UI_DESIGN_SYSTEM.md](UI_DESIGN_SYSTEM.md)，独立窗口规范见 [NATIVE_WINDOW_UI.md](NATIVE_WINDOW_UI.md)。

## 2026-05-27

### 综合面板

- 将综合面板视觉方向确认到黑 / 白 + 绿色强调。
- 支持暗黑模式，并修正暗黑主题不应偏蓝灰的问题。
- 使用 `http://127.0.0.1:4173/` 作为优先视觉预览入口，减少频繁启动 CorelDRAW。
- 综合面板继续使用 React + WebView2，保持单 WebView2 原则。

### 预览器

- 新增 NativePanelPreview 预览器，用于模拟 CorelDRAW 工具条、一级菜单、二级菜单和独立 WPF 窗口。
- 预览器支持浅色 / 暗黑切换。
- 预览器支持嵌入综合面板预览。
- 预览器阶段已满足当前视觉验证用途，后续不再优先扩展复杂预览能力。

### 工具条

- 工具条结构确认：

```text
QiTuCDR 工具条
├─ 企图插件 -> 点击直接打开综合面板，无下拉
├─ 工具分组 -> 一级菜单
│  └─ 工具项 -> 二级菜单打开独立 WPF 小窗口
```

- 移除了工具条末尾的“综合面板”一级菜单按钮。
- “企图插件”一级入口改为直接打开综合面板。
- 工具条高度已多轮微调并经用户确认达标。

### 独立 WPF 工具窗口

- 明确每个二级工具打开独立 WPF 小窗口，而不是全部在综合面板内切换。
- 独立窗口不创建 WebView2，避免违反单 WebView2 原则。
- 标题栏高度改为 32px。
- 项目 UI 全局去粗体，源码内不再使用 `Bold`、`SemiBold`、`font-weight: 500/600/700`。
- 独立窗口主外框圆角从 8px 调整为 6px。
- 独立窗口主体视觉宽度调整为 420px，外层阴影安全边距左右各 8px，WPF Window 总宽度为 436px。
- 标题栏四个控件确认：置顶、设置、展开/收起、关闭。
- 标题栏按钮热区改为矩形 hover，不使用圆角。
- 关闭按钮宽度 44px，并贴标题栏最右侧。
- 普通标题栏按钮热区为 36px × 32px。
- 标题栏图标统一为 12px × 12px，线宽 1.0px。
- 设置图标由字体符号改为 WPF 本地矢量 Path。
- 独立窗口内部布局收敛为 4px 基准网格。
- 字段标签列为 96px。
- 字段行距为 12px。
- 输入框和下拉框高度统一为 32px。
- 复选框最小高度统一为 32px。
- 描述文本行高调整为 20px。
- 面板内乱码中文已恢复为正常中文。
- 标题文本后方新增 `1.0.0` 版本标签，点击后打开窗口内部版本信息弹层。
- 设置按钮从系统 `MessageBox` 改为窗口内部设置弹层。
- 版本信息和设置弹层统一为 316px 宽、6px 圆角、右上浮层布局。
- 弹层滚动区域改为微信风格窄滚动条：透明轨道、4px 滑块、悬停 6px，不显示默认 Windows 箭头。
- 暗黑 / 浅色主题新增滚动条滑块颜色令牌。
- 标题栏弹层从窗口内部叠加层调整为独立 WPF 浮动窗口 `NativeToolPopupWindow`。
- 弹层现在可以拖动，也可以拖出独立工具窗口外；独立工具窗口收起 / 展开时不会被弹层撑开或影响自动高度。
- 弹层标题区调整为 36px 顶部整行拖动热区，解决顶部点击无法拖动的问题。
- 弹层标题文字和关闭按钮改为同一水平线居中对齐。
- 弹层滚动条改为贴右边缘，并默认隐藏；鼠标进入滚动区域后才显示。
- 弹层标题和内容区域增加 12px 间距，避免内容紧贴标题栏。
- 弹层关闭按钮改为 44px × 36px，贴右边，hover 使用危险红色，和独立 WPF 主窗口关闭按钮一致。
- 设置弹层从复选框列表改为左文本、右开关的微信式设置行。
- 弹层关闭按钮进一步收敛为 34px × 22px 小型横向矩形，避免红色 hover 背景显得过大。
- 设置弹层删除描述文字和行分隔线，只保留文本与右侧开关，并增加内容左侧内边距。
- 版本弹层和设置弹层统一使用左右 20px 内容内边距，避免不同弹层之间视觉漂移。
- 弹层关闭按钮调整为 44px × 28px，上贴边、右贴边的横向长方形热区，只保留右上角圆角，其余角为直角。
- 修正弹层滚动条覆盖正文的问题：滚动条恢复为右侧独立 8px 槽位，内容区不再被滚动条遮挡。
- 弹层关闭图标从 12px 调整为 10px。
- 弹层滚动条改为贴近外壳右边缘的浮动位置，内容区通过 12px 右侧避让避免遮挡正文。
- 弹层滚动条进一步调整为右边距 0px，贴弹层外壳右边缘；内容区右侧避让增加到 20px。
- 弹层关闭按钮红色 hover 背景高度从 28px 增加到 30px。
- 标题栏弹层新增 `ESC` 关闭。
- 标题栏弹层新增点击外部关闭，基于 WPF `Deactivated` 事件实现。
- 标题栏弹层新增当前运行会话内的位置记忆，拖动后关闭再打开会恢复上次位置。
